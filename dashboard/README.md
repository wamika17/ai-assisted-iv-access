# Dashboard

This folder contains the graphical user interface (GUI) for the Smart AI-Assisted IV Access System.

The dashboard displays patient information, physiological measurements, AI-generated vein analysis, and IV access recommendations in real time.

---

## Dashboard Features

- Patient Information
- Heart Rate
- Blood Oxygen Saturation (SpO₂)
- Vein Visibility
- DIVA Score
- Recommended Vein
- Battery Status
- Device Connection Status
