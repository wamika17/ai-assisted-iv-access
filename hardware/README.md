# Hardware

This folder contains the hardware architecture and embedded electronics of the Smart AI-Assisted IV Access System.

The hardware integrates infrared imaging, physiological sensing, and embedded processing to assist healthcare professionals during intravenous (IV) cannulation.

---

## Hardware Components

- ESP32 Microcontroller
- Infrared Camera Module
- MAX30102 Pulse Oximeter
- OLED Display
- Rechargeable Battery
- Charging Module

---

## Responsibilities

- Capture infrared vein images
- Acquire physiological parameters
- Power management
- Transfer sensor data
- Interface with the AI software
